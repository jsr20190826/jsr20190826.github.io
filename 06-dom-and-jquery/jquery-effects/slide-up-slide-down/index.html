.slideUp() & .slideDown()

.slideUp() & .slideDown() Methods

  • Methods used to hide or show the HTML elements by gradually decreasing or increasing their height

.slideUp()

Example

$(() => {
  $("button").click(() => {
    // slide #box element up with duration of 2 secs
    $("#box").slideUp(2000)
  })
})

JS Bin on jsbin.com

.slideDown()

Example

$(() => {
  $("button").click(() => {
    // slide #box element down
    $("#box").slideDown()
  })
})
